# NG-fonticon
## fonticon with a taste of Nigeria


If you need to call the font from your CSS file, use this tag : " font-family: ng-fonticon"

### 1 Download the font and unzip it

### 2 Add the CSS file “styles.css” to your project, and link to it from your HTML page

### 3 Copy the "fonts" folder into your website project

### 4 Insert the icon class name into any HTML tag to add the icon. The default CSS class prefix is 'icon-'

      <div class="icon icon-ng"></div>
